This great plugin allows you to select media files from wordpress and to bulk download it with one click!

== Description ==

You can select as many files as you want to download them as a ZIP-Archive. The archive is created in a ajax loop, so you'll never have any problems with timeout or runtime errors.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. You can find the File Downloader in the Mediamenu
